INTEX Easy Set - это, пожалуй, один из самых недорогих и популярных надувных бассейнов, который приобретается для семейного пользования. В нем легко умещаются несколько взрослых и ребятишек.

Материал чаши бассейна выполнен по технологии SUPER-TOUGH из ПВХ. Это плотный армированный материал из трех спаянных слоев: два слоя плотного винила и сетка из полиэстра. Такое сочетание наделяет материал огромным запасом прочности. Толщина стенок полотна - 0,52 мм.

Прочность, износостойкость и долговечность

Залог тому надежная конструкция (изготовленная по инновационным запатентованным технологиям SUPER-TOUGH) из 2-х слоев высококачественного плотного винила + 1 армирующий слой – сетка из особо прочного полиэстера

Монтаж за 10 минут

Один человек без труда установит его на любой горизонтальной ровной и чистой поверхности без специального оборудования, накачает при помощи насоса верхнее кольцо-поплавок. Затем он наполнит бассейн водой, под давлением которой виниловое кольцо-каркас поднимется, стенки-борта расправятся в оригинальную форму

Ремонтопригодность

Для устранения мелких повреждений, в комплектацию входит набор "самоклеек" и руководство по эксплуатации
